## 📋 Descripción del Proyecto

**TiendaPython** es una página web interactiva que simula una tienda de tecnología ubicada en **Cochabamba, Bolivia**. El proyecto fue desarrollado de forma colaborativa siguiendo un flujo de trabajo real con Git y GitHub, incluyendo ramas, commits y Pull Requests.

### ¿Qué hace la página?

- Muestra un **catálogo de 6 productos tecnológicos** con precios en bolivianos (Bs)
- Permite **agregar productos al carrito** de compras interactivo
- El carrito permite **modificar cantidades**, eliminar productos y ver el total
- Incluye un **servidor web con Flask** para servir la página desde Python
- Diseño **responsive y moderno** con colores corporativos

## 🚀 Cómo Ejecutar el Proyecto

### Requisitos

- Python 3.x instalado
- Flask instalado

### Instalación

```bash
# 1. Clonar el repositorio
git clone https://github.com/zergioal/tiendapython

# 2. Entrar a la carpeta
cd tiendapython

# 3. Instalar Flask
pip install flask

# 4. Ejecutar el servidor
python Python.py
```

### Abrir en el navegador

```
http://localhost:5000
```

El proyecto siguió la metodología de trabajo con ramas:
